# Citations and References

This directory contains all academic, industry, and standards-based references used in the PhD research project titled:

"Offensive Adversary Simulation of Critical Infrastructure Networks"

## Scope of Citations
- Peer-reviewed research papers (IEEE, ACM, Springer)
- Government & standards documents (NIST, MITRE, CISA, ENISA)
- Publicly documented cyber incidents (sanitized)
- Industrial Control Systems (ICS/OT) security research

## Usage
- All chapters, attack simulations, and lab designs reference entries from   `references.bib`
- Case studies are mapped to MITRE ATT&CK for Enterprise & ICS
- No proprietary or classified material is included

## Citation Style
- IEEE citation format
- BibTeX for LaTeX/Overleaf compatibility
